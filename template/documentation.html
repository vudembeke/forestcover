Endpoint

GET /api/forest/<forest_id>/

Description

Retrieves environmental metrics for a specific forest such as NDVI, forest area, density, change over time, and year. You can filter the results by year using query parameters.

Forest ID Reference

Forest ID Forest Name
1Mt. Kenya Forest
2Mau Forest
4Aberdare Forest
5Kakamega Forest

Parameters

Parameter Type Required Description
forest_id integer Yes ID of the forest (see Forest ID Reference)
start_year integer No Filter to include only data from this year onwards
end_year integer No Filter to include only data up to this year

Returned Fields

Field Description
ndviNormalized Difference Vegetation Index (greenness indicator)
areaForest area in square kilometers
densityForest density in percentage
changeChange in forest metrics compared to previous data (%)
yearThe year for which the data was recorded

Example Request

GET https://forest-cover-prediction.up.railway.app/api/forest/1/?start_year=2010&end_year=2020

Status Codes

  • 200 OK - Request succeeded
  • 400 Bad Request - Invalid parameters (e.g., invalid year format)
  • 404 Not Found - Forest not found
